Welcome!


You've found the homepage of the handwriter package. Keep reading to learn more about what it is, how we did it, and how to use it.


What is it?

handwriter is an R package that breaks down and analyzes handwriting samples


How does it work?

Through a step by step process explained in: Methods.


How do I use it?

A full step by step tutorial is included in the How To section.

Information (and an example) on workflow integration can be found here.


Who made it?

handwriter is a project of the Center for Statistics and Applications in Forensic Evidence (or CSAFE) at Iowa State University.
